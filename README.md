# 🛒 Sistema Web de Controle de Vendas - FirmoSports

## 📌 Sobre o Projeto
Este projeto é um sistema web desenvolvido para gerenciar pedidos e vendas de camisas esportivas.  
A aplicação foi criada com base em uma necessidade real de organizar pedidos, clientes e informações de forma prática e eficiente.

---

## 🚀 Acesse o Projeto
👉 https://enzofirmo14.github.io/Sistema-Web-de-controle-de-vendas-firmoSports/

---

## 🚀 Funcionalidades
- Cadastro de pedidos
- Organização por cliente e produto
- Registro de tamanho e personalização
- Controle de datas dos pedidos
- Gerenciamento geral das vendas

---

## 🛠️ Tecnologias Utilizadas
- Frontend: React + Vite
- Backend: Firebase
- Banco de Dados: Firestore
- Autenticação: Firebase Authentication

---

## 🧠 Aprendizados
Durante o desenvolvimento deste projeto, foram trabalhados conceitos importantes como:

- Modelagem e estruturação de banco de dados  
- Operações CRUD (Create, Read, Update, Delete)  
- Integração entre frontend e backend  
- Autenticação de usuários  
- Organização de código e boas práticas  

Além disso, este projeto contribuiu para aprofundar o uso do Firebase, tecnologia que já venho utilizando em projetos anteriores.
